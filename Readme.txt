The code implements an AI to play Tic-Tac-Toe optimally using Minimax.
There are two main files in this project: runner.py and tictactoe.py. tictactoe.py contains all of the logic for playing the game, and for making optimal moves. runner.py implements and contains all of the code to run the graphical interface for the game.
The application can be seen in action here: https://youtu.be/ovm2HooQB6w
